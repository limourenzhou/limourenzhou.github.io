  1. linux环境配置方面：

    1. 对vmware的认识，在这里需要重点介绍的是vmware网络状态设置的地方，有着几个选项，分别是：桥接模式、NAT模式、Host-only模式之类。桥接模式，就是直接连接物理网络，它拥有与主机上系统通信的能力，以及远程通信的能力。

    2. 接下来是Xshell的使用，它相当于一个终端模拟设备，

    3. 然后是linux的一些知识，安装的是CentOS 6.8版本，来源是阿里云，所以我从developer.aliyun.com网站上找到配置路径，按照操作逐步完成。

    4. 初次注册的用户不在sudoer目录里，因此配置过程中各种受限。通过谷歌，我首先找到了一些对于root用户的描述，因而我重启CentOS，然后选择使用root用户登陆，密码我输入的是最初申请的limouren用户的密码。然后打开terminal,输入visudo命令，可以打开并且编辑修改/etc/sudoers配置文件，我的电脑上是默认使用vim打开的，然后找到一行 root ALL=(ALL)ALL，在它的下一行输入limouren ALL=(ALL)ALL。其中limouren为你要添加root权限的用户的username。

    5. 关于vim的使用，a b c：切换到编辑模式 Esc：切换到阅读模式 :wq ：一个退出命令

  1. jdk的安装

    首先我找到了一个jdk-7u80-linux-x64.rpm的下载链接XXXX，然后打开Xshell，

    下载命令：wget XXXX 。wget 命令就是从链接下载文件，yum命令是从下载源下载文件。

    安装命令：rpm -qa | grep jdk 检测是否有系统原来安装的open jdk

    sudo chmod 777 jdk-7u80-linux-x64.rpm 赋予全开权限

    sudo rpm -ivh jdk-7u80-linux-x64.rpm 安装jdk

    配置命令：sudo vim /etc/profile 打开配置文件

    在最后面添加

    export JAVA_HOME=/usr/local/jdk1.7.0_71
    export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
    export PATH=$JAVA_HOME/bin:$PATH

    source /etc/profile 使配置文件生效

    java -version 可以检验是否成功安装

    1. Tomcat：Tomcat是Java EE运行的容器

    安装Tomcat之前需要安装好jdk,因为启动Tomcat需要用到jdk。

    下载 wget XXX XXX为下载路径，如果有签名限制则需要从下载内容那里复制。

    解压缩 tar -zxvf apache-tomcat-7.0.73.tar.gz

    配置环境变量 sudo vim /etc/profile

    在最后面添加 CATALINA_HOME 为Tomcat的安装目录

    保存并且使配置生效 :wq –> source /etc/profile

    配置UTF-8：给${CATALINA_HOME}/conf/server.xml文件里的Connector节点添加URIEncoding=”UTF-8”

    我也不知道，把Tomcat安装到limouren 的下属文件夹是好还是坏，反正接下来一段时间都会使用limouren这个用户登录，先这样吧。

    Tomcat 启动：${CATALINA_HOME}/bin/startup.sh

    Tomcat 关闭：${CATALINA_HOME}/bin/shutdown.sh

    上面是linux下的命令，windows则是运行那两个.bat后缀的文件

    1. Maven：Maven是Java项目的构建和管理工具

      下载 wget http://learning.happymmall.com/maven/apache-maven-3.0.5-bin.tar.gz

      脱壳 tar -zxvf apache-maven-3.0.5-bin.tar.gz

      环境变量配置：需要注意该变量中可以使用的其他变量，必须放在该变量的前面，比如：export PATH=$JAVA_HOME/bin:$PATH:$MAVEN_HOME/bin

      PATH变量里面出现了MAVEN_HOME变量，则MAVEN_HOME变量必须写在PATH前面。

      Maven常用命令：

      清除命令： mvn clean

      编译命令：mvn compile

      打包命令：mvn package

      跳过单元测试：mvn clean package -Dmaven.test.skip=true
